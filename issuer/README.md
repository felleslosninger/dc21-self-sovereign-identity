# Issuer

A SpringBoot serverapp that simulates a Issuer in a SSI-ecosystem.


## Insallation
This project requires Maven v3.6.3. To install maven:

Go to: https://maven.apache.org/download.cgi

or

```Node
npm install maven
```

## Usage
To run this project:

```Maven
mvn spring-boot:run
```
